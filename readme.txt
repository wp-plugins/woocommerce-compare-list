WooCommerce Compare List plugin adds ability to compare some products of your WooCommerce driven shop.

== Description ==

WooCommerce Compare List plugin adds ability to compare some products of your WooCommerce driven shop. Compared products are shown on a page (defined in the settings) as an aggregate table of all products attributes. The plugin builds SEO friendly URLs for compare page, which could be shared by users.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
1. Activate `WooCommerce Compare List` plugin through the 'Plugins' menu in WordPress
1. Select compare page at Compare list tab of WooCommerce settings
